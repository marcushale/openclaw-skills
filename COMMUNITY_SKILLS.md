# Community Skills

These are skills I use daily that were created by others and published on [ClawHub](https://clawhub.ai). They're listed here for reference — **install them from ClawHub directly** to get updates from their maintainers.

## Installing from ClawHub

```bash
# Browse available skills
open https://clawhub.ai

# Install a skill (check each skill's page for exact commands)
# Skills get installed to your ~/clawd/skills/ directory
```
